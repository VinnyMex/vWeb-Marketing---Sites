# Contexto do Cliente: Academia Vigor

Este arquivo é a fonte da verdade para agentes de IA que editam este site.

## 🎯 Objetivos de Negócio
- **Principal:** Captura de leads para matrículas presenciais.
- **Público-Target:** Moradores locais interessados em musculação e crossfit.
- **Proposta de Valor:** A única academia da região com acompanhamento nutricional incluso.

## 🎨 Identidade Visual (Design System)
- **Cores Primárias:** #D32F2F (Vermelho Vigor), #FBC02D (Amarelo Energia)
- **Cores Secundárias:** #212121 (Preto Fosco), #FFFFFF (Branco Limpo)
- **Tipografia:** 
    - Heading: 'Roboto Condensed', sans-serif
    - Body: 'Inter', sans-serif
- **Estilo Visual:** Energético, moderno, com alto contraste e fotos de alta resolução de pessoas treinando.

## 📝 Tom de Voz e Copy
- **Linguagem:** Motivacional, direta e inclusiva.
- **Palavras-Chave:** Superação, resultado, saúde, comunidade, vigor.
- **Termos Proibidos:** "Dieta" (usar "plano alimentar"), "Barato" (usar "acessível").

## 🛠️ Regras Técnicas Específicas
- **Integrações Ativas:** n8n (para envio de formulários de contato).
- **Restrições:** Sempre exibir o horário de funcionamento no rodapé.

---
*Instrução para o Agente: Use este contexto para decidir as cores de novos botões e o tom dos textos de chamadas para ação (CTAs).*
