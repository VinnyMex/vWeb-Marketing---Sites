# Site Exemplo - Cliente de Teste

Este é um projeto Next.js gerado automaticamente pela infraestrutura Antigravity para demonstrar o gerenciamento de sites de marketing.

## 🚀 Tecnologias
- **Framework:** Next.js 15+ (App Router)
- **Estilização:** Tailwind CSS
- **Linguagem:** TypeScript
- **Gerenciador de Pacotes:** npm

## 📂 Estrutura do Projeto
Este site reside em `_sites/site-exemplo` e está isolado dos outros projetos e da infraestrutura central.

## 🛠️ Comandos Disponíveis
- `npm run dev`: Inicia o servidor de desenvolvimento.
- `npm run build`: Cria a versão de produção.
- `npm run start`: Inicia a versão de produção.
- `npm run lint`: Verifica erros no código.
